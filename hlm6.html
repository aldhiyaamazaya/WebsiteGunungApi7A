Mitigasi Bencana Gunung Api

Kegiatan mitigasi bencana gunung api menurut PVMBG Badan Geologi :

A. Melakukan penyelidikan gunung api untuk mengetahui karakter dan sifat letusannya.

B. Pemetaan wilayah/kawasan rawan bencana bagi penduduk terhadap ancaman bahaya seperti letusan awan panas, aliran lava, aliran lahar, lontaran batu pijar, dan hujan abu.

C. Pemantauan gunung api bagi penduduk untuk mengetahui pergerakan magma dan gas.

D. Bimbingan, informasi, dan rekomendasi mengenai setiap perubahan pada gunung api yang disampaikan kepada masyarakat melalui pemerintah setempat.

E. Komunikasi yang interaktif untuk memudahkan dalam penanggulangan dan pelaporan pada setiap pengamatan gunung api secara periodik.

Ketika sebelum terjadi bencana gunung api menurut Badan Nasional Penanggulangan Bencana (BNPB), dapat melakukan hal-hal sebagai berikut :

1. Tentukan tempat pengungsian yang aman dengan mengenali daerah setempat.

2. Membuat rencana penanganan bencana.

3. Mempersiapkan pengungsian jika dibutuhkan.

4. Mempersiapkan kebutuhan dasar, seperti makanan dan minuman

Saat terjadi bencana letusan gunung api dapat melakukan hal-hal sebagai berikut :

1. Menjauhi kawasan rawan bencana seperti lereng gunung, lembah dan daerah aliran lahar.

2. Melindungi diri dari abu letusan dan awan panas jika berada ditempat terbuka.

3. Mempersiapkan diri jika kemungkinan terjadi adanya bencana susulan.

4. Mengenakan pakaian yang bisa melindungi tubuh, seperti baju lengan panjang, celana panjang, topi dan lainnya.

5. Jangan memakai lensa kontak.

6. Pakai masker atau kain untuk menutupi mulut dan hidung.

7. Ketika awan panas turun tutup wajah dengan kedua belah tangan.

Setelah terjadi letusan gunung api dapat melakukan hal-hal sebagai berikut :

1. Jauhi wilayah yang terkena hujan abu.

2. Membersihkan atap dari timbunan abu karena berat dari timbunan abu tersebut dapat merusak atau meruntuhkan atap bangunan.

3. Tidak mengendarai mobil di wilayah yang terkena hujan abu karena dapat merusak mesin motor, rem, persneling hingga pengapian.
